# Lab 

### Lab 목표 : Research Agent + MCP Tools + Analysis Agent

### Lab 시나리오

```text
User
 │
Planner Agent
 │
 ├ Research Agent
 ├ Analysis Agent
 └ Writer Agent
 │
시장 분석 보고서
```

##### Step1 : Multi-Agent 패턴 이해

```text
현재 AI Trend Automation 프로젝트에는 다음 Agent가 있다.

Research Agent
Analysis Agent
Writer Agent

이 Agent들을 기반으로 Multi-Agent 시스템을 설계하려고 한다.

다음 세 가지 Agent 패턴을 설명하고
각 패턴이 이 프로젝트에 어떻게 적용될 수 있는지 설명해줘.

1 Planner Pattern
2 Manager Pattern
3 Debate Pattern

각 패턴에 대해

- 구조
- 역할
- 장점

을 Markdown 형식으로 작성해줘.
```

##### Step2 : Multi-Agent Workflow 설계

```text
AI 산업 시장 조사 자동화 시스템을 LangGraph 기반 Multi-Agent 구조로 설계해줘.

사용할 Agent

Research Agent
Analysis Agent
Writer Agent

Workflow

Input
 ↓
Research
 ↓
Analysis
 ↓
Report

다음 내용을 포함해줘.

1 Agent 역할 정의
2 데이터 흐름
3 Workflow diagram

결과를 Markdown 문서로 작성해줘.
파일 이름은 multi_agent_workflow.md 로 만들어줘.
```

##### Step3 : LangGraph Multi-Agent 코드 생성

```text
AI Trend Automation 프로젝트를 LangGraph 기반 Multi-Agent 시스템으로 구현하려고 한다.

사용할 Agent

Research Agent
Analysis Agent
Writer Agent

LangGraph Graph 구조

User Input
 ↓
Research Node
 ↓
Analysis Node
 ↓
Writer Node

Python 코드로 구현해줘.

파일

workflow/agent_graph.py

조건

1 LangGraph 사용
2 각 Agent를 Node로 구현
3 상태(State)를 공유하도록 구현
4 코드 구조는 단순하게 유지
```

##### Step4 : Planner Agent 추가

```text
현재 Multi-Agent 시스템에 Planner Agent를 추가하려고 한다.

Planner Agent 역할

1 사용자 요청 분석
2 어떤 Agent를 실행할지 결정

새로운 구조

User
 │
Planner Agent
 │
 ├ Research Agent
 ├ Analysis Agent
 └ Writer Agent

LangGraph 기반으로 Planner Node를 추가한 코드를 작성해줘.

파일

workflow/planner_graph.py
```

##### Step5 : 시장 조사 자동화 시나리오 구현

```text
다음 시장 조사 자동화 시나리오를 Multi-Agent 시스템으로 구현해줘.

사용자 요청

AI 산업 시장 조사 보고서 작성

Agent 역할

Research Agent
→ 뉴스 및 시장 데이터 수집

Analysis Agent
→ 시장 트렌드 분석

Writer Agent
→ 보고서 작성

출력

market_analysis_report.md

LangGraph Workflow를 Python 코드로 작성해줘.
```

###### Step6 : 협업 기반 자동 실행 프로그램

```text
Multi-Agent 시스템을 실행하는 main.py를 만들어줘.

실행 순서

1 User Input
2 Planner Agent
3 Research Agent
4 Analysis Agent
5 Writer Agent

최종 결과

market_analysis_report.md 생성

코드는 가능한 단순하게 작성해줘.
```

##### Step7 : 자동 보고서 생성

```text
Multi-Agent 시스템의 최종 출력으로
AI 산업 시장 분석 보고서를 생성하도록 Writer Agent를 개선해줘.

보고서 구성

1 산업 개요
2 주요 뉴스
3 시장 트렌드

output/market_analysis_report.md
```
