# Lab 

### Lab 목표 : Research Agent + MCP Tools + Analysis Agent

### Lab 시나리오

```text
Research Agent
  │
MCP Tools
  │
Analysis Agent
  │
trend_report.md
```

##### Step1 : MCP Tool 연결

```text
현재 AI Research Agent 프로젝트에 MCP Tool을 연결하려고 한다.

다음 MCP 서버를 사용하도록 설정 파일을 생성해줘.

Tools

1 Browser MCP
2 Filesystem MCP
3 GitHub MCP

설정 파일 이름

mcp.json

Claude Code에서 사용할 수 있도록
mcpServers 설정을 JSON으로 작성해줘.
```

##### Step2 : Research Agent를 Tool 기반으로 확장

```text
현재 Research Agent는 LLM만 사용한다.

다음 Tool을 활용하도록 Research Agent를 확장해줘.

사용할 Tool

Browser
Filesystem

기능

1 Browser를 사용하여 AI 산업 뉴스 검색
2 주요 뉴스 요약
3 결과를 research_data.md 파일로 저장

파일

agents/research_agent.py

코드는 가능한 단순하게 작성해줘.
```

##### Step3 : Data Analytics Agent 생성

```
Research Agent가 수집한 뉴스 데이터를 분석하는 새로운 Agent를 만들어줘.

Agent 이름

Data Analysis Agent

입력

research_data.md

기능

1 뉴스 데이터 분석
2 주요 시장 트렌드 도출
3 핵심 기업 분석

출력

analysis_result.md

파일

agents/analysis_agent.py

Python 코드로 작성해줘.
```

##### Step4 : CSV 데이터 분석 추가

```text
Data Analysis Agent를 확장하려고 한다.

다음 기능을 추가해줘.

CSV 데이터 분석

예

ai_market_data.csv

분석 항목

1 시장 성장률
2 주요 기업 투자
3 산업 트렌드

Python에서 pandas를 사용해도 된다.

파일

agents/analysis_agent.py
```

##### Step5 : 자동 보고서 생성 Agent

```text
Research Agent와 Analysis Agent 결과를 기반으로
자동 보고서를 생성하는 Writer 기능을 만들어줘.

입력

research_data.md
analysis_result.md

출력

trend_report.md

포함 내용

1 AI 산업 뉴스 요약
2 시장 트렌드 분석
3 핵심 기업 동향
4 미래 전망

파일

agents/report_writer.py
```
